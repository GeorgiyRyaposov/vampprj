# Как добавить новое состояние

1. Создайте в этой папке новый C#-класс, реализующий интерфейс `IState` (методы `Enter`, `Exit`, `Update`).

2. В конструктор передайте `GameStateMachine` (и при необходимости другие параметры: имя сцены, сервисы).

3. В `Enter()` выполните вход в состояние (загрузка сцены, подписки, инициализация). Для перехода в другое состояние вызовите `_machine.EnterState(new ДругоеСостояние(...))`.

4. В `Exit()` выполните очистку (отписки, выгрузка, сохранение).

5. В `Update()` — логика каждый кадр, пока состояние активно.

6. Чтобы перейти на новое состояние из кода (например по кнопке):  
   `ServiceLocator.Instance.Get<GameStateMachine>().EnterState(new ВашеСостояние(...));`
